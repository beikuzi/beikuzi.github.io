# 翻页样式集合

这里包含了多种二次元风格的翻页组件样式，你可以根据需要选择使用。

## 样式列表

### 1. **sakura.css** - 樱花风格
- **特点**: 粉色系，柔和可爱
- **适合**: 少女系、温馨风格的博客
- **配色**: 粉红色渐变，白色按钮

### 2. **candy.css** - 糖果风格
- **特点**: 多彩活泼，充满活力
- **适合**: 活泼、欢快的博客主题
- **配色**: 彩虹渐变，动态闪烁效果

### 3. **starry.css** - 星空风格
- **特点**: 紫色系，梦幻神秘
- **适合**: 神秘、梦幻主题的博客
- **配色**: 深紫色渐变，星空背景动画

### 4. **dreamy.css** - 梦幻风格
- **特点**: 蓝紫色渐变，柔和梦幻
- **适合**: 少女系、梦幻风格的博客
- **配色**: 蓝紫色渐变，柔和光晕效果

### 5. **tech.css** - 科技风格
- **特点**: 霓虹灯效果，赛博朋克风格
- **适合**: 科技、游戏、赛博朋克主题
- **配色**: 青色和紫色霓虹，扫描线动画

### 6. **kawaii.css** - 可爱风格
- **特点**: 圆润可爱，粉蓝配色
- **适合**: 萌系、可爱风格的博客
- **配色**: 粉蓝色渐变，星星装饰

### 7. **magic.css** - 魔法风格
- **特点**: 金色和紫色，神秘感
- **适合**: 奇幻、魔法主题的博客
- **配色**: 紫色和金色渐变，旋转光晕

### 8. **minimal.css** - 简约风格
- **特点**: 简洁现代，但保持二次元感
- **适合**: 清新、简约风格的博客
- **配色**: 淡粉色，简洁设计

## 使用方法

在 HTML 的 `<head>` 部分引入你想要的样式：

```html
<!-- 例如使用樱花风格 -->
<link rel="stylesheet" href="../assets/css/pagination-styles/sakura.css">
```

或者在 `style.css` 中导入：

```css
@import url('./pagination-styles/sakura.css');
```

## 预览建议

建议在浏览器中逐个测试这些样式，选择最符合你博客主题的风格。
